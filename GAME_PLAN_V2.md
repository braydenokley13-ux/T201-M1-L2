# NBA GM Crisis Manager - Game Transformation Plan (v2)

## Overview
Transform the Excel-based NBA Luxury Tax simulator into an interactive HTML game where students manage REAL 2026 NBA teams with actual rosters, real trade targets, and authentic salary cap situations. Each scenario uses a different team facing unique challenges.

## CLAIM CODE SYSTEM (UPDATED)
- **Claim codes are ONLY awarded when you hit ALL win conditions** (wins target, performance points target, under spend max)
- **No claim code for partial success** - you must fully solve the scenario
- Format: L2-201-M1-[CODE]
- Displayed on results screen ONLY if you win
- Used for grading/verification

## Technical Implementation

### Tax Calculation Logic
```javascript
function calculateLuxuryTax(payroll, taxThreshold = 165, isRepeat = false) {
  const overAmount = payroll - taxThreshold;
  if (overAmount <= 0) return 0;

  // Progressive tax brackets
  const brackets = [
    { limit: 5, rate: isRepeat ? 1.5 : 1.0 },
    { limit: 10, rate: isRepeat ? 1.75 : 1.25 },
    { limit: 15, rate: isRepeat ? 2.0 : 1.5 },
    { limit: 20, rate: isRepeat ? 2.25 : 1.75 },
    { limit: Infinity, rate: isRepeat ? 3.25 : 2.5 } // Second apron
  ];

  let tax = 0;
  let remaining = overAmount;
  let floor = 0;

  for (const bracket of brackets) {
    const ceiling = bracket.limit === Infinity ? remaining : bracket.limit;
    const taxableInBracket = Math.min(remaining, ceiling - floor);
    tax += taxableInBracket * bracket.rate;
    remaining -= taxableInBracket;
    floor = ceiling;
    if (remaining <= 0) break;
  }

  return tax;
}
```

### Performance Points Calculation
```javascript
function calculatePerformancePoints(wins, playoffWins, bonuses) {
  const regWinWeight = bonuses.regWinWeight || 1;
  const playoffWinWeight = bonuses.playoffWinWeight || 3;
  const fiftyWinBonus = wins >= 50 ? (bonuses.fiftyWinBonus || 5) : 0;
  const sixtyWinBonus = wins >= 60 ? (bonuses.sixtyWinBonus || 8) : 0;

  return (wins * regWinWeight) +
         (playoffWins * playoffWinWeight) +
         fiftyWinBonus +
         sixtyWinBonus;
}
```

### Win Condition Check
```javascript
function checkWinCondition(final, targets) {
  const winsHit = final.wins >= targets.wins;
  const perfHit = final.perfPoints >= targets.perfPoints;
  const spendOk = (final.payroll + final.tax) <= targets.maxSpend;

  return {
    won: winsHit && perfHit && spendOk,
    winsHit,
    perfHit,
    spendOk,
    claimCode: (winsHit && perfHit && spendOk) ? scenario.claimCode : null
  };
}
```
